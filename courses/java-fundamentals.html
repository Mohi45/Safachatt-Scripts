What is Java?

Java is a programming language and a platform. It is high-level, robust, object-oriented, and secure.

Developed by Sun Microsystems (now Oracle) in 1995. James Gosling is the father of Java. Earlier called Oak.

Java is 99% object-oriented. Primitive types like int, float, and double make up the remaining 1%.

JRE, JDK & JVM

JRE (Java Runtime Environment)

Package to run compiled Java programs. Cannot create new programs.

JDK (Java Development Kit)

Includes JRE + compiler + tools to create programs.

JVM Memory Structure

Heap Area

Stores actual objects.

Scanner sc = new Scanner(System.in);

Stack Area

Thread-specific memory for local variables and references.

Method Area

Stores class-level data.

Native Stack & PC Registers

Used for native methods and instruction addresses.

Garbage Collection

JVM automatically cleans up unused memory.

First Java Program

class Simple {
  public static void main(String args[]) {
    System.out.println("Hello Java");
  }
}

Java Variables

A variable is a container that holds data.

  • Local: Declared inside a method, accessible only there.
  • Instance: Declared inside class, outside method, non-static.
  • Static: Declared with static keyword, shared across all instances.
public class A {
  static int m = 100; // static variable
  void method() {
    int n = 90; // local variable
  }
  public static void main(String args[]) {
    int data = 50; // instance variable
  }
}

Java Keywords

Java keywords are reserved words with predefined meanings. Examples:

  • abstract – declare abstract classes or methods
  • assert – for assertions
  • boolean – true/false values
  • break – exit loop or switch
  • byte – 8-bit integer
  • … (all other keywords can be added similarly)

throw vs throws

throw: explicitly throws an exception.

throws: declares that a method may throw an exception.

While Loop Example

int n = 10;
while (n > 0) {
  System.out.println("tick " + n);
  n--;
}
