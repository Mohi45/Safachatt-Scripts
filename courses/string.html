1Ô∏è‚É£2Ô∏è‚É£ Interview Questions + Answers (Hinglish) ‚ñº

Q1: String Pool kya hota hai?

A: String Pool ek memory area hai jaha same literal reuse hota hai memory save karne ke liye.

Q2: String immutable kyu hoti hai?

A: Security aur thread safety ke liye + hashing performance improve hoti hai.

Q3: equals() aur == me difference?

A: == reference compare karta hai, equals() content compare karta hai.

Q4: new String("Java") me kitne objects banenge?

A: 2 objects banenge (1 String Pool me + 1 Heap me).

Q5: StringBuilder aur StringBuffer difference?

A: Builder fast hai but thread safe nahi, Buffer thread safe hai but slow.

Q6: intern() kya karta hai?

A: intern() String Pool wala reference return karta hai.
